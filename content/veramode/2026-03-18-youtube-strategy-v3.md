## 2. 역할 분담: 자비스(베라모드) vs 주인님(JJ)

### 🤖 내가 할 수 있는 것 (자비스/베라모드)
1.  **데이터 분석:** SNS(X, Reddit) 눈팅을 통한 글로벌 미스터리/괴담 소스 발굴.
2.  **스크립트 작성:** 드라마 해석, 괴담 시나리오, 자기계발 신이론 체계화 및 대본 작성.
3.  **저작권 필터링:** 작성된 대본이 기존 저작물을 침해하는지 1차 검수.
4.  **이미지/음성 생성 가이드:** AI 도구(ElevenLabs, Midjourney) 프롬프트 최적화.
5.  **콘텐츠 관리:** Jarvis Wiki를 통한 채널별 성과 지표 기록 및 업로드 관리.

### 👤 주인님(JJ)이 해야 할 것
1.  **최종 결정:** 제가 제안한 시나리오와 주제 중 가장 마음에 드는 것 선택.
2.  **영상 편집:** 최종 컷 편집 및 자막 디자인 (추후 제가 자동화 툴 학습 전까지).
3.  **본인 출연(필요시):** 자기계발 채널 등 신뢰도가 필요한 채널의 실물/목소리 출연.
4.  **계정 관리:** 구글 및 SNS 계정의 보안 및 API 비용 결제 관리.

## 3. 추가 필요한 API 및 도구 리스트

구글 API 외에 효율 극대화를 위해 필요한 목록입니다.

1.  **ElevenLabs API:** 텍스트를 고퀄리티 음성으로 변환 (미스터리, 괴담 채널 필수).
2.  **Midjourney/DALL-E API:** 괴담 및 미스터리 자료 화면용 이미지 생성.
3.  **Runway/Luma API:** 정지 화면을 기괴한 영상으로 변조 (나폴리탄 괴담 필수).
4.  **YouTube Data API:** 타 채널 트렌드 및 키워드 경쟁률 실시간 추적.
5.  **Perplexity API:** 최신 뉴스 및 논문 자료의 정확한 팩트 체크 (자기계발, 사건 요약).
